Search and retrieve movie details from the IMDb database and store it in the local database when user selects the movie. Only for the first time, movie details are crawled from IMDb site. Next time, movie details will be served from the local database. Two different methods are used to retrieve search results.

Api Search: This method uses the IMDb search api to retrieve search results and use that information to crawl through the website for more info.

Web Crawl Search: This method retrieves the search result by sending input to the search bar and use that information to crawl through the website for more info.

Using Api search is more efficient than Web Crawling. Web Crawling need atleast 20 seconds to retrieve the suggestions for single query. Test it yourself!!!
